How to  start this project ?

open your terminals,

1.For Backend
cd backend,
npm install, for all dependencies,
make sure you added .env file, and add them,
PORT = 5000,
DB = 'your-database',
SECRET = 'add-your-jwt-secret-any-name',
npm start for start this project
1.For Frontend
cd frontend,
npm install, for all dependencies,
npm start for start this project
